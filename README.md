## 🌟 프로젝트 소개 (Introduction)

**예인교회 공식 웹사이트**는 성도님들과의 원활한 소통과 복음 전파를 위해 현대적이고 반응형으로 설계된 웹 서비스입니다. 
Next.js의 최신 기술을 활용하여 빠른 로딩 속도와 최적화된 사용자 경험을 제공하며, Contentful CMS를 연동하여 실시간으로 말씀과 소식을 업데이트합니다.

🔗 **라이브 사이트**: [www.yeinchurch.kr](https://www.yeinchurch.kr)

---

## ✨ 주요 기능 (Key Features)

### 📹 말씀/신앙생활 (Faith & Sermons)
- **최신 설교 영상**: 유튜브 API와 연동된 실시간 설교 업데이트 및 아카이브
- **새벽 묵상/칼럼**: 매일 제공되는 영성 깊은 말씀과 신앙 칼럼
- **온라인 주보**: 매주 업데이트되는 주보를 PDF 및 이미지로 제공

### 🖼️ 교회 소식 (Church News)
- **교회 갤러리**: 예인교회의 소중한 순간들을 담은 고화질 이미지 갤러리
- **팝업 공지**: 중요한 교회 행사 및 공지사항을 알리는 유연한 팝업 시스템

### 🏢 교회 소개 (About Us)
- **예배 안내**: 상세한 예배 시간표 및 오시는 길(지도 연동)
- **부서 소개**: 각 소그룹 및 부서별 활동 안내
- **상담 센터**: 성도님들의 고민을 나누는 온라인 상담 연동

## 🛠 기술 스택 (Tech Stack)

### Frontend
- **Framework**: Next.js (App Router)
- **Styling**: Vanilla CSS (CSS Modules)
- **State Management**: React Hooks

### Backend & Infrastructure
- **CMS**: Contentful (Headless CMS)
- **Deployment**: Vercel
- **Version Control**: Git & GitHub

## 🖥️ 실행 방법 (Local Development)

```bash
# 의존성 설치
npm install

# 환경 변수 설정 (.env.local 파일 작성 필요)
# CONTENTFUL_SPACE_ID=your_id
# CONTENTFUL_ACCESS_TOKEN=your_token

# 개발 서버 실행
npm run dev
```
